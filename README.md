# Daily Reflection Tree Assignment

This repository contains my submission for the DT Fellowship Assignment.

I designed a deterministic Daily Reflection Tree that guides users through an end-of-day reflection process using structured tree data and a working Python agent. The system uses fixed-choice questions, predefined branching logic, and reflection summaries without relying on any LLM at runtime.

The reflection flow is built across three psychological axes:

1. **Ownership** – internal control vs external blame
2. **Contribution** – value creation vs entitlement
3. **Awareness** – self-focus vs broader perspective

## Repository Contents

* **tree/** → Reflection tree files (TSV / JSON) and tree diagram
* **agent/** → Runnable Python program that executes the tree
* **write-up.md** → Design decisions, logic, and improvements

## Objective

The goal of this project is to transform human reflection into a predictable, auditable, and meaningful system that promotes self-awareness, contribution mindset, and wider thinking.
